# Attendance Tracker (AWS Cloud Project)

A cloud-based attendance management system built using Python, Flask, and AWS Elastic Beanstalk.

# Features
- Admin authentication
- Add and search student attendance
- Attendance percentage analytics
- Cloud deployment using AWS

# Tech Stack
- Python
- Flask
- SQLite (/tmp cloud-safe storage)
- AWS Elastic Beanstalk
